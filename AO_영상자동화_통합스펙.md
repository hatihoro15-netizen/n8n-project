## 2. 절대 규칙

1. **프롬프트 100% 보존**: P1(원문 지침)은 변경/요약/의역/삭제 절대 금지
2. **이미지 슬롯 자동 삽입**: 이미지 1~4 → IMG_1~IMG_4 슬롯에 자동 매핑
3. **프롬프트가 영상 내용을 결정**:
   - 사람 사진 넣고 "개구리가 노래하는 영상" → 사진 무시, 개구리 영상 생성
   - 사람 사진 넣고 "이 사람이 축구하는 영상" → 사진 활용, 축구 영상 생성
4. **구글 시트 사용 금지**: 상태/로그 전부 DB
5. **한글 입력 기본 지원**: 한글만 넣어도 동작 + 영문 번역 자동 생성
6. **자동 재시도**: AI API 실패 시 3회 자동 재시도

## 4. 이미지/영상 소스 (3가지)

1. **사용자 직접 업로드** — 이미지 1~4장, 레퍼런스 영상
2. **AI 이미지 생성** — 프롬프트로 이미지 생성 (Flux Pro/SDXL/Ideogram)
3. **인터넷/유튜브** — 관련 이미지/영상 검색해서 가져오기

## 5. 입력 스키마

| 필드 | 필수 | 설명 |
|------|------|------|
| prompt_p1 | Y | 원문 지침 (절대 변경 금지) |
| translate_mode | N | 한글→영문 처리 (auto/on/off) |
| topic | Y | 주제 (1줄 정의) |
| keywords | Y | 키워드 (쉼표 구분) |
| category | Y | 카테고리 |
| images[1..4] | N | 이미지 1~4 (슬롯 자동 삽입) |
| ref_video | N | 레퍼런스 영상 |
| use_media | Y | 미디어 모드 (auto/forced/off) |
| template_id | N | 영상 템플릿 선택 (뉴스/광고/스토리 등) |
| upload_target | Y | YouTube |
| metadata | N | 제목/설명/태그 |

### use_media 모드
- **auto**: 이미지/영상 있으면 활용, 없으면 프롬프트만으로 생성
- **forced**: 제공된 이미지/영상 반드시 함께 사용
- **off**: 프롬프트만 사용 (미디어 무시)

### 파생 필드 (자동 생성)
- prompt_lang_detected: 언어 감지 결과
- prompt_en: 영문 복사용 프롬프트
- prompt_to_engine: 실제 엔진에 전달된 최종 프롬프트

## 6. AO 프롬프트 조립기

P1을 재작성하지 않고, 100% 유지한 채 변수만 슬롯에 주입:
- {TOPIC}, {KEYWORDS}, {CATEGORY}
- {IMG_1}~{IMG_4}, {REF_VIDEO}
- (선택) {LANG}, {TONE}, {DURATION}, {ASPECT_RATIO}

품질 기준:
- P1 텍스트는 원문 그대로 포함
- 사용자 입력은 지정 슬롯에만 삽입 (슬롯 외 임의 삽입 금지)
- 누락된 값은 빈 문자열/NULL 처리, 로그에 기록

## 8. n8n 아키텍처 (Producer / Worker / Queue)

### Producer (작업 등록, 병렬 가능)
1. 입력 수신 (웹훅/폼)
2. 필수값 검사
3. Job 생성 (job_id, status=queued)
4. Queue에 Push
5. 응답 (job_id 반환)

### Worker (작업 처리, 1건씩)
1. Queue에서 Pop (1건)
2. AO 프롬프트 조립 → 최종 실행 프롬프트
3. AI 이미지 생성 (필요 시)
4. TTS 생성
5. Creatomate 영상 렌더링 (템플릿 적용)
6. AI 썸네일 생성 (선택)
7. YouTube 업로드 (OAuth)
8. 상태 업데이트 + 로그 기록

### Job 상태
queued → processing → generated → uploading → uploaded
↘ failed (3회 자동 재시도 → retrying → processing)

### n8n Webhook 엔드포인트
| 트리거 | URL | 데이터 |
|--------|-----|--------|
| 이미지 생성 | /webhook/generate-image | 모델, 프롬프트, 개수 |
| 나레이션 생성 | /webhook/generate-script | 주제, 유형, 톤, AI 모델 |
| 영상 제작 | /webhook/make-video | 이미지, 스크립트, 목소리, 템플릿 |
| 업로드 | /webhook/upload-video | 영상, 제목, 예약시간 |
