**절대 금지:** 단독 자음/모음 전부 (ㅋㅋ, ㅎㅎ, ㅠㅠ 등), 한 글자짜리 감탄
**대체:** 웃음→비꼬는 톤 / 슬픔→"아...", "하..." / 혀차기→"이런", "참나" / 놀람→"헐"
**숫자:** 아라비아 숫자 금지. 한글로 풀어쓸 것.

## ★★★ dialogue 필드 화자 구분 규칙 (Veo3 음성 배분용) ★★★
**dialogue 필드에 반드시 화자 이름을 붙일 것.**
- 형식: "화자이름: 대사 화자이름: 대사"
- 화자 표기: **Jay** / **Mike**

## 출력 직전 검증
1. 6턴 순서: 장면1(B→A) / 장면2(B→A) / 장면3(A→B) 맞는가?
2. 각 장면이 정확히 2턴씩인가?
3. 4턴이 2문장 브릿지 구조인가?
4. 장면1에 금지어 19개 없는가?
5. 4턴에 "스포츠 배팅"/"분석"/"배당률" 직접 단어 있는가?
6. "온카스터디" 5턴에서만 1회 등장하는가?
7. 블랙리스트 패턴 사용하지 않았는가?
8. 단독 자음/모음 없는가? 아라비아 숫자 없는가?
9. image_prompt에 프로필 전문이 축약 없이 포함되었는가?
10. **dialogue에 "Mike:" / "Jay:" 화자 태그가 붙어있는가?**
11. **이전 에피소드와 4턴 브릿지 방식, 5턴 Flex 방식이 다른가?**
