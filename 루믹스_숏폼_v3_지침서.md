## 3. 단계별 상세 설명

### 3.1 트리거
| 노드명 | 타입 | 설명 |
|--------|------|------|
| 스케줄 트리거 | Schedule Trigger | cron: `0 3,15 * * *` (매일 03시, 15시 UTC) |
| Webhook Trigger | Webhook | POST `/webhook/lumix-v3-trigger` 수동 실행용 |
| Manual Trigger | Manual | n8n UI에서 직접 실행 |

### 3.2 분석 리포트 읽기
- **타입**: Google Sheets (GET)
- **문서**: n8n LUMIX 시트
- **시트**: 분석리포트
- **목적**: 채널 성과 분석 데이터를 읽어 AI 주제 생성에 반영
- **데이터 필드**: 트렌드_키워드, 추천_주제1~3, 피해야할_주제, 효과적인_훅_팁, 고성과_패턴
- **참고**: 데이터가 없어도 fallback 처리됨 ('데이터 없음')

### 3.3 AI 주제 생성 (1차)
- **모델**: Gemini 2.5 Flash Lite
- **프롬프트 구성**:
  1. 분석 리포트 데이터 삽입 (트렌드, 추천주제, 피해야할 주제 등)
  2. 루믹스 솔루션 서비스 정보 (통합 솔루션, API 공급, 독점 플랫폼, 라이브 데모)
  3. 광고 스타일 5종 중 랜덤 선택 (대화형/후기형/정보형/비교형/트렌드형)
  4. 숏폼 최적화 규칙 (후킹, 루프 구조, 감정 흐름)
- **출력 JSON**:
  ```json
  {
    "Subject": "영상 제목 (30자 이내)",
    "Narration": "나레이션 전문 (80~150자)",
    "Caption": "YouTube 설명 (해시태그 포함)",
    "Comment": "첫 댓글 (공감 유도)",
    "BGM_prompt": "BGM 분위기 영어 설명"
  }
  ```

### 3.4 주제 파싱
- **타입**: Code (JavaScript)
- **기능**:
  - Gemini 응답에서 JSON 추출
  - 나레이션 150자 초과 시 문장 단위로 자르기
  - Status: '준비', generatedAt 타임스탬프 추가

### 3.5 AI 검증 시스템
- **검증 횟수**: 최대 2회 (3차는 검증 생략)
- **평가 항목** (각 1~10점, 합계 60점 만점):
  1. 훅 파워 - 첫 문장 스크롤 멈춤 효과
  2. 주제 관련성 - 트렌드/추천 주제 부합도
  3. 나레이션 품질 - 5문장 구성, 80~150자, 자연스러움
  4. 클릭 유도력 - 제목의 호기심 유발
  5. 타겟 적합도 - 카지노 솔루션 운영자/사업자 대상
  6. 차별화 - 독창적 각도
- **통과 기준**: 42점 이상 (7점 평균)
- **흐름**:
  - 1차 통과 → 바로 시트 기록
  - 1차 탈락 → 피드백 반영 2차 생성 → 2차 검증
  - 2차 통과 → 시트 기록
  - 2차 탈락 → 3차 생성 (검증 생략, 파이프라인 중단 방지)

### 3.6 시트 기록

### 3.11 NCA 영상 합성 (FFmpeg)

#### NCA 데이터 준비
- **핵심 로직**: 11개 소재를 분석하여 FFmpeg compose API payload 생성
- **식별 방법**:
  - video.url 있으면 → 영상 파트
  - audio.url + metadata.duration → BGM
  - audio.url + file_size → 나레이션 (file_size로 duration 추정)
- **FFmpeg 처리**:
  - 각 영상: scale 1080x1920 + stream_loop + trim
  - 각 나레이션: atrim으로 duration 맞춤
  - concat 필터로 5파트 연결
  - BGM: volume 0.30 (30%) + amix로 믹싱
- **API 형식** (NCA Toolkit):
  ```json
  {
    "inputs": [{"file_url": "...", "options": [{"option": "-flag"}, {"option": "value"}]}],
    "outputs": [{"options": [{"option": "-filter_complex"}, {"option": "필터문자열"}, ...]}],
    "global_options": []
  }
  ```

#### NCA 영상 제작
- **URL**: `http://76.13.182.180:8080/v1/ffmpeg/compose`
- **Body**: `{{ JSON.stringify($json.nca_payload) }}`
- **결과**: 합성된 영상 URL 반환

#### NCA 제작 결과
- NCA API 응답에서 영상 URL 추출
- 응답 형식: `{response: [{file_url: "..."}]}` 또는 `{response: "url"}`

## 5. 트러블슈팅

### 자주 발생하는 에러

| 에러 | 원인 | 해결 |
|------|------|------|
| `'inputs' is a required property` | NCA API에 잘못된 형식 전송 | inputs/outputs/global_options 구조 확인 |
| `Unrecognized option 'stream_loop -1'` | FFmpeg 옵션 합쳐서 전송 | 각 옵션을 별도 object로 분리 |
| `video_url is not of type 'object'` | 문자열 배열을 object 배열로 전송 필요 | `{"video_url": "..."}` 형태 |
| fal.ai 잔액 부족 | TTS/이미지/BGM 생성 실패 | fal.ai 대시보드에서 충전 |
| Gemini rate limit | 무료 플랜 20 RPM 초과 | 잠시 대기 후 재실행 |
| MinIO 403 Forbidden | 버킷 권한 비공개 | `mc anonymous set download nca/nca-toolkit` |
| video_url 빈 값 | NCA API 응답 파싱 실패 | response[0].file_url 체크 |

## 6. 콘텐츠 가이드라인

### 루믹스 솔루션 핵심 서비스
1. **통합 솔루션** - 올인원 비즈니스 패키지 (마케팅, 유저 관리, 운영 전반)
2. **API & 알 개별 공급** - 글로벌 본사 직연동, 중간 벤더 제거, 업계 최저 요율
3. **독점 플랫폼 제작** - 커스텀 UI/UX, 풀스택 개발, 차별화된 브랜딩
4. **라이브 데모** - 실제 운영 환경과 동일한 데모 제공

### 숏폼 제작 규칙
- **길이**: 30~60초
- **나레이션**: 80~150자, 5문장, 문장마다 마침표/물음표/느낌표
- **구조**: 3초 후킹 → 약속 → 본문 → 루프 연결/CTA
- **루믹스 언급**: 나레이션 후반부 1~2회만, 자연스럽게
- **BGM**: 영상 분위기에 맞는 영어 프롬프트 (10~20단어)
- **루프 구조**: 마지막 문장이 첫 문장과 자연스럽게 연결

### 광고 스타일 (5종 랜덤)
1. 대화형 - 두 사람의 문제→해결
2. 후기형 - 실제 경험담
3. 정보형 - 충격적 사실 전달
4. 비교형 - Before/After
5. 트렌드형 - 최신 트렌드 + 솔루션 연결

## 8. 미배포 수정사항 (fix_all_v3.py)

아직 적용되지 않은 5가지 개선사항:
1. **BGM 볼륨**: 0.30 → 0.10 (30% → 10%)
2. **자막 위치**: 중앙 → 하단 (y=h-200)
3. **영상 길이**: +1초 버퍼 → +2초 버퍼
4. **영상 슬로우모션**: stream_loop 반복 → setpts 슬로우다운 + tpad 프레임 유지
5. **자막 정확도**: 자동 음성 인식 → drawtext (실제 나레이션 텍스트 사용)

배포 방법: `python3 /tmp/fix_all_v3.py` 실행 후 워크플로우 업데이트
