# 유튜브 롱폼 제작 완전 가이드

---

## 1. 핵심 지표 (YouTube Studio에서 확인)

### 평균 시청 지속 시간 (Average View Duration) ⭐ 가장 중요
- 시청자가 평균적으로 몇 분/초를 봤는지 (시간 단위)
- 이 수치가 높을수록 알고리즘이 강하게 추천
- 롱폼에서는 이 지표가 채널 성장을 좌우

### 평균 조회율 (Average Percentage Viewed)
- 영상 전체 길이 대비 몇 %를 봤는지 (비율 단위)
- 10분 영상에서 평균 5분 시청 = 50%
- ※ 스튜디오에서 위 두 지표가 별도로 표시됨

### CTR (클릭률)
- 피드에서 썸네일/제목 보고 클릭한 비율
- 전체 채널/동영상 중 절반은 CTR 2%~10% 사이
- 채널 규모/유입원(추천 vs 검색)에 따라 기준이 다름
- ⚠️ CTR이 높아도 초반 이탈이 크면 노출이 급격히 꺾임

### 시청 지속 시간 (Watch Time)
- 영상 전체에서 누적된 총 시청 시간
- 수익화 조건(4,000시간)에 직결
- 롱폼의 가장 큰 강점 = 시청시간 축적

### 참여 신호
- 좋아요 / 댓글 / 공유 / 저장
- 구독 전환 (이 영상을 보고 구독이 늘었는지)
- 세션 유지 (내 영상 → 다음 영상으로 이어짐)

### 노출수
- 영상이 추천/검색 결과에 얼마나 노출되었는지
- 노출 대비 CTR을 함께 분석

### 목표 설정
**목표는 고정값이 아니라 '채널 평균 대비 개선'을 원칙으로 하고, '정보형/리스트형/스토리형'에 따라 다르게 잡는다.**

| 지표 | 참고 기준 |
|------|----------|
| 초반 30초 유지율 | 70% 이상 (콘텐츠 타입별 조정) |
| 평균 조회율 | 40~50% 이상 |
| CTR | 채널 평균 대비 개선 |
| 평균 시청 지속 시간 | 영상 길이의 40%+ |

## 2. 영상 길이 전략

| 길이 | 용도 | 특징 |
|------|------|------|
| 3~5분 | 튜토리얼/팁 | 짧고 집중된 정보 전달 |
| 8~12분 | 정보/리뷰/분석 | 광고 삽입 최적 (미드롤 8분+), 가장 범용적 |
| 15~25분 | 심층 분석/다큐 | 시청시간 대량 축적, 충성 시청자 확보 |
| 30분+ | 강의/팟캐스트/라이브 | 니치 시장, 높은 충성도 필요 |

### 길이 결정 기준
- 콘텐츠가 요구하는 만큼만 (억지로 늘리지 않기)
- **8분 이상이면 미드롤 광고 삽입 가능** → 수익에 직접 영향
- 정보 밀도가 떨어지면 차라리 짧게 편집

## 3. 영상 구성 전략

### 오프닝 구조 (첫 30초가 승부를 가른다)

| 구간 | 역할 | 예시 |
|------|------|------|
| 0~3초 | 훅 (결론/충격/궁금증) | "이 한 가지 때문에 90%가 실패합니다" |
| 3~10초 | 약속 (이 영상에서 얻는 것) | "오늘 3가지 핵심 전략을 알려드리겠습니다" |
| 10~30초 | 증거/예시로 신뢰 확보 | "실제 데이터를 보여드리겠습니다" |

→ 이 구조를 지키면 초반 이탈이 크게 줄어듭니다.

⚠️ **절대 하면 안 되는 것:** 긴 인트로 애니메이션, "안녕하세요 OO입니다" 로 시작, 영상과 관련 없는 잡담

### 본문 구성
- 스토리텔링 방식으로 전개 (기승전결)
- **리텐션 포인트:** 2~3분마다 시각적 변화 or 새로운 정보 투입
- 불필요한 부분 과감히 컷 편집 (점프컷 활용)
- "잠깐, 여기서 중요한 게 있는데..." 같은 주의 환기 멘트
- B-roll, 그래픽, 텍스트 오버레이로 시각적 다양성

### 중반 이탈 방지 기법
- 오픈 루프: "이건 나중에 말씀드릴게요" → 궁금증 유지
- 패턴 인터럽트: 갑자기 화면 전환, 효과음, 톤 변화
- 진행 표시: "첫 번째는...", "두 번째로..." → 구조 예측 가능
- 챕터로 구간 나누기

### 엔딩
- 핵심 내용 요약 (30초 이내)
- 다음 영상 예고 또는 관련 영상 추천
- 구독/좋아요/알림 CTA
- 엔드스크린 카드 (관련 영상 2개 추천)

### 챕터 기능 (타임스탬프)
- 설명문에 타임스탬프 추가 → 시청자 편의 + SEO 효과
- Google 검색 결과에도 챕터별로 노출 가능
- 형식: `0:00 인트로 / 1:23 첫 번째 팁 / 3:45 두 번째 팁`

## 4. 썸네일 & 제목

### 썸네일
- 제목과 썸네일이 일치해야 일관된 메시지 전달
- 명확한 주제 전달 + 시각적 임팩트
- 사람 얼굴·강한 감정 표현이 주목도를 올리는 경향
- 3명 이상 포함된 이미지는 주목도 감소 경향
- 텍스트는 3~5단어 이내, 큰 글씨
- 유튜브 스튜디오 공식 A/B 테스트 기능 활용
- ※ A/B 테스트는 계정/영상 조건에 따라 제공 범위가 다를 수 있음 (스튜디오에서 노출될 때 사용)

### 제목
- 같은 영상도 제목에 따라 CTR 2배 이상 차이
- 인기 유튜버(100만+ 구독자)의 제목 패턴 분석
- 경쟁 영상 상위 5개의 제목과 조회수 비교
- 60자 이내로 핵심 메시지 전달
- 숫자, 질문형, 궁금증 유발 키워드 활용
- **제목과 썸네일이 같은 말을 반복하면 안 됨** → 서로 보완하는 관계

## 5. 검색 키워드 (SEO)

### 키워드 리서치 도구
- TubeBuddy: 심층 키워드 연구, 높은 트래픽 키워드 발굴
- VidIQ: 최적 키워드 탐색, 검색 순위 향상 도구
- Google Trends: 트렌드 키워드 확인
- YouTube 검색 자동완성: 실시간 인기 검색어 파악
- Ahrefs/SEMrush: 경쟁 분석 (유료)

### 키워드 적용 위치
- 제목: 핵심 키워드를 앞쪽에 배치
- 설명: 첫 2~3줄에 주요 키워드 포함 + 상세 설명
- 태그: 광범위한 태그 + 세부 태그 함께 사용
- 자막/CC: 검색 엔진이 내용을 인식하는 데 도움
- 파일명: 업로드 전 파일명에도 키워드 포함 (예: casino-solution-review.mp4)

### 경쟁자 분석
- 유사 영상 상위 5개의 제목, 설명, 태그 확인
- 조회수와 게시 시기 분석
- 댓글에서 시청자가 원하는 콘텐츠 파악
- 경쟁 영상의 시청 지속 시간 그래프 패턴 참고

### 설명문 작성법
- 첫 2~3줄: 핵심 키워드 + 영상 요약 (검색 결과에 노출됨)
- 타임스탬프 (챕터)
- 관련 링크 / SNS / 사이트
- CTA (구독, 좋아요, 다음 영상)
- 해시태그 3~5개
- 총 200~500자 권장

## 6. 업로드 최적화

### 업로드 시간
- YouTube Studio → 분석 → 시청자 → "시청자가 YouTube를 사용하는 시간" 확인
- 시청자가 가장 활발한 시간대 1~2시간 전에 업로드
- 일반적 골든타임: 평일 오후 2~4시, 주말 오전 9~11시 (타겟에 따라 다름)

### 업로드 주기
- 주 2~3회 권장 (최소 주 1회)
- 일정한 주기가 알고리즘에 긍정적
- 불규칙한 업로드는 알고리즘 노출에 불리
- 요일별 고정 업로드 시간 설정 추천

### 카드 & 엔드스크린
- 카드: 영상 중간에 관련 영상/플레이리스트 추천
- 엔드스크린: 마지막 20초에 관련 영상 2개 + 구독 버튼
- 플레이리스트: 관련 영상끼리 묶어 세션 유지 극대화

## 7. 수익화

### 수익화 조건 (2025~2026년 공식 기준)

**확장(초기) YPP:**
- 구독자 500명
- 최근 90일 내 유효 공개 업로드 3개
- 지난 12개월 유효 공개 시청시간 3,000시간 또는 최근 90일 유효 쇼츠 조회수 300만

**전체(광고수익 공유) YPP:**
- 구독자 1,000명
- 지난 12개월 유효 공개 시청시간 4,000시간 또는 최근 90일 유효 쇼츠 조회수 1,000만

⚠️ **중요:** 쇼츠 피드에서 발생한 시청시간은 4,000시간 요건에 포함되지 않음

### 롱폼 수익의 강점
- 미드롤 광고 (8분+ 영상): 수익 크게 증가
- CPM이 숏폼 대비 훨씬 높음
- 프리롤 + 미드롤 + 포스트롤 = 영상 1개당 광고 여러 개

### CPM 높은 카테고리
- 금융, 비즈니스, 기술 튜토리얼: CPM $15~25
- 교육, 부동산: CPM $10~20
- 엔터테인먼트, 게임: CPM $2~6

### 광고 외 수익화 방법
1. 브랜드 협찬/스폰서십
2. 제휴 마케팅 (설명문에 링크)
3. 자체 상품/서비스 판매
4. 채널 멤버십
5. 슈퍼챗/슈퍼땡스
6. YouTube Premium 수익
7. 온라인 강의/컨설팅 유도

## 8. 저작권 & 재사용 콘텐츠 체크 ⚠️

### 체크 항목
- [ ] BGM 라이선스 확인 (상업적 사용 가능 여부)
- [ ] 영상 소스 라이선스 확인 (스톡 영상/이미지)
- [ ] 인용/캡처 시 공정 이용 범위 확인
- [ ] AI 생성 콘텐츠 출처/표기/정책 변화 대비

### 재사용 콘텐츠 리스크
- 단순 짜깁기(편집 거의 없음)는 "재사용 콘텐츠"로 판정
- AI 생성 영상이라도 고유한 편집, 나레이션, 텍스트 등 독창성 필수
- **재사용 콘텐츠 평가는 채널 전체를 보고 판단 → 제작/참여/편집 흔적이 명확해야 안전**
- YouTube 정책은 계속 변하므로 정기적으로 확인

### Content ID 대응
- 타인의 음악/영상 사용 시 Content ID 클레임 가능
- 클레임 = 해당 영상 수익이 원저작자에게 이전
- 반복 시 저작권 경고 → 채널 삭제 위험
- YouTube 오디오 라이브러리 무료 음원 활용 추천

## 9. 분석 & 개선

### 매일 확인
- CTR (클릭률)
- 평균 시청 지속 시간
- 실시간 조회수 변화
- 댓글/반응
- 좋아요/공유/저장 수

### 주 1회 확인
- 시청자 이탈 그래프 분석 (어디서 빠지는지)
- 경쟁 채널 분석
- 트렌드 키워드 업데이트
- 시청자 활동 시간대
- 트래픽 소스 (검색, 추천, 외부 등)
- 구독 전환율 (영상별)

### 월 1회 확인
- 채널 전체 성장 추이
- 가장 성과 좋은 영상 유형 분석
- 수익 분석 (CPM, RPM)
- 콘텐츠 전략 재점검
- 저작권/정책 변화 확인

### 시청자 이탈 그래프 읽는 법
- 초반 급락: 훅/썸네일-콘텐츠 불일치
- 중반 서서히 하락: 정보 밀도 부족, 지루한 구간
- 특정 지점 급락: 해당 구간 내용/편집 문제
- 끝까지 유지: 좋은 구성 → 이 패턴을 반복
