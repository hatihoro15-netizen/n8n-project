# 유튜브 숏폼(Shorts) 제작 완전 가이드

---

## 1. 핵심 지표 (YouTube Studio에서 확인)

### 완주율 (Completion Rate) ⭐ 가장 중요
- 영상 끝까지 본 시청자 비율
- 숏폼에서 알고리즘이 가장 크게 보는 지표
- 목표: 채널 평균 대비 지속적 개선 (일반적으로 60%+ 권장)

### 재시청률 (Replay Rate)
- 같은 영상을 반복 시청한 비율
- 재시청이 높으면 알고리즘이 강하게 밀어줌
- 루프 구조가 좋으면 자연스럽게 올라감

### CTR (클릭률)
- 피드에서 썸네일/제목 보고 클릭한 비율
- 채널 규모/유입원(추천 vs 검색)에 따라 기준이 다름
- 자기 채널 평균 대비 개선을 목표로 할 것

### 세션 유지 (Session Watch)
- 내 쇼츠를 본 후 다음 영상(내 영상 or 유튜브 내 영상)으로 이어지는지
- 시청자가 유튜브에 오래 머물게 하면 알고리즘이 보상

### 참여 신호
- 좋아요 / 댓글 / 공유 / 저장
- 구독 전환 (이 영상을 보고 구독이 늘었는지)

### 목표 설정
**목표는 고정값이 아니라 '채널 평균 대비 개선'을 원칙으로 하고, 콘텐츠 타입에 따라 다르게 잡는다.**

| 지표 | 참고 기준 |
|------|----------|
| 완주율 | 60% 이상 |
| 초반 3초 유지율 | 90% 이상 |
| CTR | 채널 평균 대비 개선 |
| 재시청 | 높을수록 유리 |

## 2. 영상 길이 전략

| 길이 | 용도 | 특징 |
|------|------|------|
| 15~45초 | 초반 폭발형 | 임팩트 있는 한 가지 포인트, 빠른 템포 |
| 45~75초 | 정보/스토리형 | 구조화된 내용 전달, 밸런스 좋음 |
| 1~3분 | 서사/설명형 | 깊이 있는 내용, 확장성·수익성 높음 |

- 유튜브는 쇼츠를 최대 3분까지 지원 (세로 영상 규격 요건 충족 시)
- 핵심은 길이보다 **완주율·재시청·세션 유지**
- 짧은 쇼츠도 훅/루프 구조가 좋으면 여전히 터질 수 있음
- **1분 이상 쇼츠는 확장성과 수익 면에서 점점 유리해지는 추세**

---

## 3. 영상 구성 전략

### 첫 1~3초: 훅 (가장 중요)
- 시청자가 스크롤을 멈추게 만드는 순간
- 결론 먼저 / 충격적 장면 / 강한 질문 / 반전 예고
- 예시: "이거 모르면 손해입니다", "99%가 틀리는 문제"

### 3~10초: 약속
- "이 영상에서 알 수 있는 것" 명확히 제시
- 시청자가 "끝까지 봐야겠다"고 느끼게 만들기

### 10초~끝: 본문
- 정보 밀도를 높여 이탈 방지
- 불필요한 공백/전환 없이 빠른 템포 유지
- 시각적 변화 (텍스트, 이펙트, 화면 전환) 2~3초마다

### 마지막 1~2초: 루프 or CTA
- **루프 구조:** 끝이 자연스럽게 처음으로 연결 → 재시청 유도
- **CTA:** "팔로우하면 더 알려드릴게요" / "댓글로 알려주세요"
- 루프와 CTA 중 콘텐츠에 맞는 것 선택

### 루프 구조 만드는 법
- 영상 마지막에 처음과 같은 장면/키워드 배치
- 결말이 곧 시작이 되는 순환 구조
- "아까 말한 거 기억나죠?" 같은 연결 멘트

---

## 4. 제목 & 해시태그

### 제목
- 짧고 강렬하게 (30자 이내 권장)
- 궁금증 유발 or 결론 힌트
- 숫자 활용: "3가지", "1분만에", "90%가"
- 이모지 적절히 활용 (1~2개)

### 해시태그
- #Shorts 태그는 더 이상 필수 아님 (세로 영상이면 자동 인식)
- 관련 해시태그 3~5개
- 너무 광범위한 태그보다 니치 태그가 효과적
- 예시: #카지노솔루션 #온라인카지노 #API인증

### 설명문
- 핵심 키워드 포함 (1~2줄이면 충분)
- CTA 한 줄 추가

## 5. 제작 스펙

### 필수 규격
| 항목 | 스펙 |
|------|------|
| 화면 비율 | 9:16 (세로) |
| 해상도 | 1080x1920 권장 |
| 최대 길이 | 3분 |
| 파일 형식 | MP4 권장 |
| 프레임 레이트 | 30fps 이상 |

### 자막/텍스트
- **필수:** 무음 시청자 70%+ (소리 없이 보는 사람 많음)
- 화면 상단 1/3에 핵심 텍스트 배치
- 하단은 댓글/좋아요 UI에 가려질 수 있으므로 주의
- 폰트: 굵고 읽기 쉬운 것, 배경 대비 높은 색상

### BGM/효과음
- 트렌딩 오디오 사용 시 노출 증가 가능
- 효과음으로 강조 포인트 만들기
- 저작권 확인 필수

---

## 6. 검색 키워드 (SEO)

### 도구
- TubeBuddy: 키워드 연구, 태그 최적화
- VidIQ: 키워드 탐색, 경쟁 분석
- YouTube 검색 자동완성: 실시간 인기 검색어

### 경쟁자 분석
- 같은 니치의 인기 쇼츠 상위 5개 분석
- 제목, 해시태그, 댓글 반응 확인
- 조회수 대비 좋아요/댓글 비율 체크

---

## 7. 업로드 최적화

### 업로드 시간
- YouTube Studio → 분석 → 시청자 활동 시간 확인
- 시청자가 가장 활발한 시간대 1~2시간 전에 업로드

### 업로드 주기
- 매일 1개 이상 권장
- 일정한 주기가 알고리즘에 긍정적
- 한 번에 몰아서 올리기보다 꾸준히 분산

### 배치 제작 팁
- 하루에 5~10개 분량 한꺼번에 촬영/편집
- 예약 업로드 활용
- 자동화 워크플로우 활용 시 품질 관리에 신경

---

## 8. 수익화

### 수익화 조건 (2025~2026년 공식 기준)

**확장(초기) YPP:**
- 구독자 500명
- 최근 90일 내 유효 공개 업로드 3개
- 지난 12개월 유효 공개 시청시간 3,000시간 또는 최근 90일 유효 쇼츠 조회수 300만

**전체(광고수익 공유) YPP:**
- 구독자 1,000명
- 지난 12개월 유효 공개 시청시간 4,000시간 또는 최근 90일 유효 쇼츠 조회수 1,000만

⚠️ **중요:** 쇼츠 피드에서 발생한 시청시간은 4,000시간 요건에 포함되지 않음

### 쇼츠 수익 구조
- 크리에이터 풀 방식: 전체 쇼츠 광고 수익을 조회수 점유율로 배분
- 쇼츠 광고 수익 자체는 롱폼 대비 낮음 (RPM 평균 $0.01~0.06)
- 수익을 키우려면 물량(볼륨)으로 보완

### 광고 외 수익화 방법
1. 브랜드 협찬/스폰서십 (높은 조회수 = 협상력)
2. 제휴 마케팅 (설명문/댓글 고정에 링크)
3. 자체 상품/서비스 판매 유도
4. 롱폼 영상으로 유도 → 롱폼에서 수익화
5. 슈퍼 땡스

## 9. 저작권 & 재사용 콘텐츠 체크 ⚠️

### 체크 항목
- [ ] BGM 라이선스 확인 (상업적 사용 가능 여부)
- [ ] 영상 소스 라이선스 확인
- [ ] 이미지/밈 사용 시 저작권 확인
- [ ] AI 생성 콘텐츠 출처/표기/정책 변화 대비

### 재사용 콘텐츠 리스크 (자동화 채널 특히 주의)
- 단순 짜깁기(편집 거의 없음)는 "재사용 콘텐츠"로 판정
- AI 생성 영상이라도 고유한 편집, 나레이션, 텍스트 등 독창성 필수
- **재사용 콘텐츠 평가는 채널 전체를 보고 판단 → 제작/참여/편집 흔적이 명확해야 안전**
- YouTube 정책은 계속 변하므로 정기적으로 확인

## 10. 분석 & 개선

### 매일 확인
- 완주율 / 재시청률
- 실시간 조회수 변화
- 좋아요/댓글/공유/저장 수

### 주 1회 확인
- 가장 성과 좋은 쇼츠 유형 분석
- 경쟁 채널 인기 쇼츠 분석
- 트렌드 키워드/오디오 업데이트
- 구독 전환율 (영상별)

### 월 1회 확인
- 채널 전체 성장 추이
- 수익 분석 (RPM)
- 콘텐츠 전략 재점검
- 저작권/정책 변화 확인

---

## 11. 업로드 전 체크리스트

### 콘텐츠
- [ ] 첫 1~3초 훅이 강력한가?
- [ ] 정보 밀도가 충분한가? (빈 구간 없는가?)
- [ ] 루프 구조 또는 CTA가 있는가?
- [ ] 텍스트 자막이 들어갔는가?
- [ ] 화면 하단 UI 가림 영역을 피했는가?

### 패키징
- [ ] 제목이 짧고 궁금증을 유발하는가?
- [ ] 해시태그 3~5개 넣었는가?

### 기술
- [ ] 9:16 세로 화면인가?
- [ ] 1080x1920 해상도인가?
- [ ] 3분 이내인가?

### 저작권
- [ ] BGM/영상/이미지 라이선스 확인했는가?
- [ ] 재사용 콘텐츠 판정 리스크 없는가?
- [ ] AI 생성 콘텐츠 정책을 준수하는가?

### 업로드
- [ ] 최적 업로드 시간에 예약했는가?
- [ ] 카테고리/언어 설정이 맞는가?

---

핵심 요약: 숏폼은 **첫 1~3초 훅 + 완주율 + 재시청**이 전부다. 이 3가지를 극대화하는 데 모든 제작 에너지를 집중하라.
